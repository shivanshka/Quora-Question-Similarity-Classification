About Application

This application helps to identify semantically and contextually similar questions. It is very usefull for platforms like Quora to make their service more efficient and optimized. This will help them to detect similar question and provide answer (already available answer in there database). In this there are two modes Single and File Mode, in File Mode you can upload a csv file and it will return file with predictions. In Single Model, it will only compare two questions inputs.
